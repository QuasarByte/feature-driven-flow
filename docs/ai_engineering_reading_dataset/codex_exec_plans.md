# Codex Execution Plans
Source: https://developers.openai.com/cookbook/articles/codex_exec_plans

## What Is an ExecPlan
An execution plan is a structured plan used by coding agents to complete large tasks.

## Typical Structure
- Objective
- Context
- Milestones
- Validation

## Benefits
- Encourages planning before coding
- Improves agent reliability
- Enables multi‑hour autonomous work
