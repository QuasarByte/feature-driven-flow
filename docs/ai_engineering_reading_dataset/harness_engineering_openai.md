# Harness Engineering: Leveraging Codex in an Agent‑First World
Source: https://openai.com/index/harness-engineering/

## Overview
OpenAI describes building an internal beta product where almost all code was generated by AI agents.
Engineers focused on guiding agents rather than manually writing code.

## Key Ideas
- Humans specify intent
- Agents generate and modify code
- Repository documentation becomes critical
- Architecture constraints are enforced mechanically
