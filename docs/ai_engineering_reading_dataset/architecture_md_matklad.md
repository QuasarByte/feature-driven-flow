# ARCHITECTURE.md
Source: https://matklad.github.io/2021/02/06/ARCHITECTURE.md.html

## Purpose
An `ARCHITECTURE.md` file explains the high‑level design of a codebase.

It helps new contributors understand where to make changes.

## Guidelines
- Keep the document short
- Focus on stable architectural ideas
- Provide a code map of important modules
- Document architectural invariants

## Example
The rust‑analyzer project contains a well‑known example of this style.
