# AI Is Forcing Us To Write Good Code
Source: https://bits.logic.inc/p/ai-is-forcing-us-to-write-good-code

## Thesis
AI coding tools work best when codebases follow good engineering practices.

## Practices That Help AI
- Automated tests
- Clear architecture
- Good documentation
- Modular design

## Implication
Best practices are no longer optional—they directly improve AI productivity.
